Web Technology Course Curriculum

Bold text: This course provides a comprehensive introduction to modern web development.
Italic text: Students will learn foundational concepts of front-end and back-end programming.
Underline text: **Prerequisite:** Basic computer literacy is required.

Unordered List: Key Learning Outcomes

  • Design and structure web pages using HTML.
  • Style web pages using CSS.
  • features with JavaScript.
  • responsive design principles.

Ordered List: Weekly Schedule

  1. Introduction to HTML5 Structure
  2. CSS3 Styling
  3. Layout Techniques (Flexbox and Grid)
  4. JavaScript Fundamentals
  5. Working with Forms

Description List: Course Modules

Front-End Development
Focuses on the visual and interactive aspects of a website that users directly interact with.
Back-End Development
Deals with the server-side logic, databases, and application programming interfaces (APIs).
Full-Stack Development
The ability to work on both front-end and back-end parts of an application.

Nested List: Unit Topics

  • Unit 1: HTML Fundamentals
    1. Basic Tags and Structure
    2. Links and Images
    3. Semantic HTML5 Elements
  • Unit 2: CSS Essentials
    • Selectors and Specificity
    • Box Model
    • Typography and Color

Mixed List: Special Activities

  • Required: Final Project Submission (Deadline: Week 12)
    1. "HTML & CSS: Design and Build Websites"
    2. "JavaScript: The Definitive Guide"
